6. cv2操作视频

   - 视频由多张图片组成 + 音频

   - 操作视频代码

     ```python
     # 如果参数是0，则调取本地的摄像头
     cap = cv2.VideoCapture(0)
     # cap = cv2.VideoCapture('./datas/video1.mp4')
     
     detector = cv2.CascadeClassifier('./datas/haarcascade_frontalface_default.xml')
     
     while cap.isOpened():
         flag, frame = cap.read()
         gray = cv2.cvtColor(frame, code=cv2.COLOR_BGR2GRAY)
         face_zone = detector.detectMultiScale(gray, scaleFactor=1.2, minNeighbors=5)
         
         for x, y, w, h in face_zone:
             cv2.circle(frame, center=(x + w//2, y + h//2), radius=w//2, color=[0,0,255], thickness=2)
         
         if flag == False:
             break
         
         cv2.imshow('temp', frame)
         
         if ord('q') == cv2.waitKey(5):
             break
     
     cv2.destroyAllWindows()
     cap.release()
     ```

## 9. 机器学习

- 什么是机器学习

### 线性回归

- 最小二乘法：主要思想是选择未知参数，使得理论值和观测值之差的平方和达到最小

  ![在这里插入图片描述](https://i-blog.csdnimg.cn/direct/3b29993f84274732ac1453aea30d1740.png)

  ![在这里插入图片描述](https://i-blog.csdnimg.cn/direct/356196ef1982494bb4760574f47a9a92.png)

  $w=X^TX^{-1}X^Ty$
  
- `from sklearn.linear_model import LinearRegression`

  - `fit_intercept=False`：不求截距

    > 如果有截距，就不能用最小二乘法求w了，得用梯度下降法求w和b

  - 什么是梯度：梯度就是导数，导数是函数变化率最快的方向

  - 梯度下降是求解最大值、最小值最快的路径

### Ridge回归

- 岭回归，对线性回归的优化，增加了一个惩罚项（二阶正则），α越大，收缩量就越大，系数对共线性越稳定。所以岭回归的目的：**将w尽量变成0，即删除不重要的特征**。
  $$
  \min_{w} \|Xw - y\|_2^2 + \alpha \|w\|_2^2
  $$

- 解决：特征比样本还多的问题。因为输入的数据矩阵X不是满秩矩阵，非满秩矩阵在求逆的时候会出现问题。

- 岭回归就是在矩阵 $X^TX$ 上加一个 $\lambda I$ 从而使得矩阵非奇异，进而能对 $X^TX+\lambda I$ 求逆。通过引入该惩罚项，能够减少不重要的参数，这个技术在统计学中也叫做缩减（shrinkage）。

- 正则化：防止过拟合，提高泛化能力。

- 过拟合：系数特别多，系数绝对值很大。

### Lasso回归

- 套索回归，估计稀疏系数的线性模型，使用了l1正则化（一阶正则）。
  $$
  \min _ { w } \frac { 1 } { 2 n _ { \mathrm { s a m p l e s } } } \| X w - y \| _ { 2 } ^ { 2 } + \alpha \| w \| _ { 1 }
  $$

- 

- 系数比较大，减少系数的绝对值，让w往0靠，使网络中的权重尽可能为0，也就相当于减小了网络复杂度，防止过拟合。
- PS：当属性很多时，选择lasso

### 逻辑回归

- Logistics，解决的是 **分类** 问题。

- 所有的分类问题都是概率问题。

- sigmoid函数
  $$
  \sigma ( z ) = { \frac { 1 } { 1 + e ^ { - z } } }
  $$

- sigmoid函数的导数：
  $$
  \sigma' ( z ) = { \frac { e ^ { - z } } { (1 + e ^ { - z }) ^ 2 } } = \sigma(z)(1- \sigma(z))
  $$
  
- 把线性回归的结果缩放到 0-1 之间完成分类。
  $$
  h _ { \theta } ( x ) = g ( \theta ^ { T } x ) = { \frac { 1 } { 1 + e ^ { - \theta ^ { T } x } } }
  $$

### 决策树

- 用的是 **信息论**，逻辑回归用的是最大似然（概率），贝叶斯用的也是概率论

- 香农 Shannon 引入物理学中的 **熵**，描述信息的混乱程度。
  $$
  H = - \sum _ { i = 1 } ^ { n } p ( x _ { i } ) \log _ { 2 } p ( x _ { i } )
  $$

- Ross Quinlan发明了：

  - ID3：信息增益

  - C4.5：信息增益率

  - C5.0：使用更少的内存和更少的规则集，更准确

  - CART：分类+回归，gini指数，**二叉树**
    $$
    H(X_m)=\sum_k p_{mk}(1-p_{mk})
    $$

- 决策树的深度：max_depth=1 是值两层，深度是1，**树的剪枝**

### 随机森林/极端决策树

- 由多棵决策树组成，集成学习
- 随机：训练集抽样、特征抽样
- **极端决策树**，ExtraTrees，其实也是一种极端的随机森林，直接用随机阈值，而不是在每个🌲中选择最优阈值。
- 属于Bagging、套袋法，在原始训练集的随机子集上构建黑盒估计其的若干实例，然后聚合它们的各个预测以形成最终预测

### Adaboost

- 在重复 **修改的数据** 版本上拟合一系列弱学习器，然后加权多数投票以产生最终预测。预测错的数据权重增加，预测正确的数据权重减少。
- 第一个弱学习器：每个类中数据的 **权重** 为各个类样本的 **占比**
- `from sklearn.ensemble import AdaBoostClassifier`

### GBDT

- 梯度提升数
- 以前向阶段构建加法模型，允许优化任意可微分损失函数，每个阶段的弱学习器拟合偏差损失函数的负梯度。
- 学习残差，改进结果，最终结果=弱学习器的预测加和

### XGBoost

- 中国人，陈天奇，后面几十个博士维护升级修改优化
- 优化的分布式梯度增强库，对gbdt的升级
- 一般情况下，xbg效果好于gbdt
