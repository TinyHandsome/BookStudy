# scikit-learn使用fetch_mldata无法下载MNIST数据集的问题

## 0. 写在前面

**参考书**

《Python数据科学手册》

**工具**

python3.5.1，Jupyter Lab

## 1. 问题描述

如题，显示下载超时，链接不上，不能下载等。

## 2. 解决办法

直接下载MNIST数据集。

<https://github.com/amplab/datascience-sp14/raw/master/lab7/mldata/mnist-original.mat>

然后把数据集保存到mldata文件夹中：
